1. Создать базу данных PostgreSQL и указать данные для подключения в файле Config.

2. Запустить приложение с помощью команды `flask run` в терминале.

3. В браузере открыть адрес `http://localhost:5000/` для просмотра главной страницы. 

4. Для просмотра записей логов нужно перейти по адресу `http://localhost:5000/logs`.

5. Для получения данных через API нужно отправить GET-запрос на адрес `http://localhost:5000/api/logs`. По запросу с параметрами start_date и end_date будут возвращены записи за указанный временной промежуток, а по запросу с параметром ip будут возвращены записи с заданным IP-адресом.
